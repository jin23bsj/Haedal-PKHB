# 🌟 Dream Achiever - Flutter 프론트엔드

## 📁 폴더 구조
```
lib/
├── main.dart                  ← 앱 진입점 + 바텀 네비게이션
├── theme/
│   └── app_theme.dart         ← 색상 & 테마 (파스텔)
├── models/
│   ├── goal.dart              ← 목표 데이터 모델
│   └── daily_record.dart      ← 일일 기록 모델
├── services/
│   ├── api_service.dart       ← ⭐ 백엔드 URL 설정 여기!
│   ├── goal_service.dart      ← 목표 API 호출
│   ├── record_service.dart    ← 기록 API 호출
│   └── chat_service.dart      ← 챗봇 API 호출
├── providers/
│   ├── goal_provider.dart     ← 목표 상태관리
│   └── record_provider.dart   ← 기록 상태관리
└── screens/
    ├── home_screen.dart       ← 홈 (streak, 오늘 기록, 목표 현황)
    ├── goal_list_screen.dart  ← 목표 목록 (진행중/완료)
    ├── goal_form_screen.dart  ← 목표 추가/수정
    ├── record_screen.dart     ← 오늘 기록 입력
    ├── analysis_screen.dart   ← 성장 분석 + 차트
    └── chat_screen.dart       ← Gemini AI 챗봇
```

## 🚀 시작하는 법

### 1. 패키지 설치
```bash
flutter pub get
```

### 2. 백엔드 URL 연결
`lib/services/api_service.dart` 파일에서:
```dart
static const String baseUrl = 'http://YOUR_BACKEND_URL/api';
//                                    ↑ 여기를 실제 주소로 바꾸세요!
```

### 3. 앱 실행
```bash
flutter run
```

## ⚠️ 백엔드 팀원에게 확인할 것

백엔드 팀원에게 꼭 물어보세요:
- API 주소 (로컬 개발 중이면 보통 `http://10.0.2.2:8080/api` - 안드로이드 에뮬레이터 기준)
- 각 API 엔드포인트 이름 (예: `/goals` vs `/goal`)
- 요청/응답 JSON 필드 이름 (카멜케이스 vs 스네이크케이스)
- 인증 필요 여부 (JWT 토큰 등)

## 🎨 화면 구성

| 탭 | 화면 | 기능 |
|---|---|---|
| 🏠 홈 | HomeScreen | streak, 오늘 기록 여부, 목표 현황, 감정 히스토리 |
| 🎯 목표 | GoalListScreen | 목표 목록, 추가/수정/삭제, 달성률 |
| ✏️ 기록 | RecordScreen | 감정 선택, 행동 기록, 메모 |
| 📊 분석 | AnalysisScreen | 달성률 차트, 감정 변화 그래프, 과거 vs 현재 |
| 💬 챗봇 | ChatScreen | Gemini AI 대화 |
